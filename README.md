Samave S.L.

-Desarrollo integro del proyecto: Diseño, Programación y Despliegue.
-Objetivo Principal: Crear una plataforma actualizada, rápida y accesible, añadiendo funcionalidad responsive, para mejorar posicionamiento de la empresa en internet y mejorar la visibilidad de cara al cliente.

-Se Utiliza:
  -HTML5/CSS3/JavaScript Vanilla ES6
  -PHP 8 (Base para posible futura integración de formulario)
  -Despliegue en hosting Apache.
  -Conversión de imagenes a formato .webp y compresión nativa.

-Logros:
  -SEO: Usando PageSpeed optimizamos el contenido y código para mejorar el posicionamiento
  -Diseño Responsivo: La desfasada web no era válida para navegar a través de dispositivos moviles, en esta nueva versión si lo es.
